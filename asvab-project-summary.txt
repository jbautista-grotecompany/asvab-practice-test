PROJECT OVERVIEW
----------------
Built a fully functional, standalone ASVAB Math Practice Test web page hosted
on GitHub Pages. The test covers both math sections of the ASVAB — Arithmetic
Reasoning (AR) and Mathematics Knowledge (MK) — with 38 verified questions
across 15 topics, styled in Chicago White Sox colors with a watermark, Google
Analytics tracking, and automatic email delivery of results.

LIVE URL
--------
https://jbautista-grotecompany.github.io/asvab-practice-test/

FILE DELIVERED
--------------
index.html — single self-contained file, no dependencies except Google Fonts
and the GA4 script tag. No framework, no build step, works in any browser.

================================================================================
FEATURES
================================================================================

1. WHITE SOX THEME
   - Black background (#0d0d0d) with silver (#c4c4c4) accents
   - Subtle diagonal pinstripe overlay mimicking Sox jersey
   - EB Garamond serif font for headings for a classic/classy feel
   - Inter sans-serif for body and UI elements
   - Full dark mode — colors defined with CSS variables throughout

2. WATERMARK
   - Fixed bottom-right corner: "Created by Jacinto Bautista"
   - Italic EB Garamond, semi-transparent silver
   - Non-interactive (pointer-events: none)

3. STICKY SCORE TRACKER (header)
   - Live score display: correct / answered
   - Progress bar fills as questions are completed
   - "X of 38 done" counter

4. QUESTION STRUCTURE
   - Questions grouped by topic with topic labels
   - Each card shows section tag (AR or MK) and question number
   - After answering: correct answer turns green, wrong answer turns red,
     correct answer is also revealed in green if wrong choice was made
   - Explanation shown below after every answer

5. INTERACTIVE GRAPHS (coordinate geometry only)
   - "Show graph" button appears in explanation after answering
   - Slope question: plots both points, line, and dashed rise/run triangle
     with labels "rise = 8, run = 4"
   - Y-intercept question: plots y = 3x − 5 with y-intercept highlighted
   - Point-on-line question: plots y = 2x + 1 with all 4 answer choices
     as colored dots (correct = white, wrong = red)
   - Graphs drawn with HTML5 Canvas, no external libraries

6. RESULTS MODAL
   - Appears automatically when all 38 questions are answered
   - Shows total score, percentage, AR score, and MK score separately
   - Silently submits results to Formspree on completion

7. RESET BUTTON
   - Resets all question cards and score tracker to initial state

================================================================================
QUESTION AUTHENTICITY
================================================================================

All questions were researched and written to match real ASVAB phrasing and
difficulty based on official sources including:
  - officialasvab.com (official ASVAB sample questions)
  - Kaplan ASVAB Total Prep
  - ASVAB for Dummies (Rod Powers)
  - asvabpracticetests.com
  - asvabadvantage.com

Key design decisions:
  - Absolute value questions use PEMDAS-inside-bars format (e.g. |4×3 − 20|)
    matching how the real ASVAB presents them, not simple |−n| lookups
  - Probability uses multi-color marble bags requiring total calculation first
  - No calculator allowed on ASVAB — all arithmetic kept manageable by hand
  - AR questions use plain civilian scenarios (not military-specific) to match
    the actual test format

All 38 answers were programmatically verified before final delivery.

Bugs caught and fixed during QA:
  1. Q30 (point on line): options (2,5) and (3,7) were BOTH valid answers for
     y=2x+1. Fixed by changing (2,5) to (2,6) so only (3,7) is correct.
  2. Cylinder volume question: had a garbled explanation with contradictory
     text left over from an earlier fixedAns patch. Rewritten cleanly.
  3. Absolute value questions: original phrasing (|-15+4|) was valid but weak.
     Replaced with authentic ASVAB-style questions using PEMDAS inside the bars.
